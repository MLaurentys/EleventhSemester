NOTEWORTHY:

  • Use of bulma for almost all layout classes, guaranteeing mobile friendlyness. All the additional layout style is in default-styles and is tested to be mobile friendly as well. Font and sizing styles are found in shadow doms.
  • When you load the app page, it is very likely that you will bump into cards without an image. I could remove those, but that would break the game in some cases, making it impossible to reach target.
  • The game is beatable in most cases, but cases it is not are not handled.
  • There is support for choosing start and end points to provide customized play, but also to test the use of localStorage.
  • The options part of the app is loaded in a single request... I could not find a way to paginate the request.
